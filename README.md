# Portfolio Sample

A sample portfolio website designed with HTML, CSS and JavaScript.

## Overview

This project showcases a modern portfolio website built with fundamental web technologies. It demonstrates clean design principles and interactive functionality to create an engaging user experience.

## Technologies Used

- **HTML** (37.6%) - Structure and markup
- **CSS** (53.8%) - Styling and layout
- **JavaScript** (8.6%) - Interactivity and dynamic features

## Features

- Responsive design
- Clean and modern user interface
- Interactive elements with JavaScript
- Professional portfolio showcase

## Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/riteshsingha/portfolioSample.git
```

2. Navigate to the project directory:
```bash
cd portfolioSample
```

3. Open `index.html` in your web browser:
```bash
open index.html
# or simply double-click the index.html file
```

## Usage

This portfolio site can be used as:
- A personal portfolio template
- A reference for web design and development
- A starting point for your own portfolio website
